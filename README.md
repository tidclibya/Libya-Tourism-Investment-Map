# 🗺️ الخارطة الاستثمارية السياحية لليبيا

هذا المشروع يقدم خريطة تفاعلية لجميع **المواقع السياحية المستهدفة للاستثمار** في ليبيا، بناءً على بيانات رسمية من الهيئة العامة للسياحة (إدارة تنمية وتطوير المناطق السياحية).

## 📊 البيانات تشمل:
- **145 موقعاً** موزعة على 13 شعبية.
- **المواقع المخرطة** (تم إضفاء الصبغة السياحية عليها بموجب قرارات رسمية).
- **المواقع المحددة للتخريط** (مقترحة للتخطيط المستقبلي).
- المساحة، القرار، الملاحظات، والحالة.

## 🔍 الميزات:
- خريطة تفاعلية (Leaflet + OpenStreetMap).
- تصفية حسب المنطقة والحالة (تم تخريطه / محدد للتخريط).
- مربع بحث بالاسم.
- جدول بيانات سفلي قابل للطي.
- إحداثيات تقديرية ذكية لتوزيع النقاط على الخريطة.

## 📁 ملفات المشروع:
- `index.html` – الواجهة الكاملة (HTML/CSS/JS).
- `sites.csv` – قاعدة البيانات.
- `README.md` – هذا الشرح.

## 🚀 التشغيل عبر GitHub Pages:
1. ارفع الملفات إلى مستودع عام.
2. فعّل إعداد **GitHub Pages** على فرع `main` المجلد الجذر.
3. افتح الرابط: `https://tidclibya.github.io/Libya-Tourism-Investment-Map/`

- يمكن استبدالها بإحداثيات حقيقية عند توفرها.
